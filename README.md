# MusicTab

### * Requirements

- Node.js runtime
- NestJS framework (backend)
- Yarn or npm
- TypeScript
- Prisma ORM (for DB)

### * Run backend
```bash
npx ts-node --project apps/backend/tsconfig.json -r tsconfig-paths/register apps/backend/src/main.ts
```

### * Run tests
```bash
yarn vitest run
```
